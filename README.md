# Skin-Care-Inventory-System

## Overview
A terminal-based inventory and sales management system developed in Python for managing skin care products.

## Features
- Display available products
- Purchase products
- Restock inventory
- Automatic invoice generation
- Timestamped transaction records
- Inventory updates after every transaction

## Technologies
- Python
- File Handling
- Modular Programming

## How to Run
python Main.py

## Learning Outcomes
- File handling
- Functions
- Modular design
- Error handling
